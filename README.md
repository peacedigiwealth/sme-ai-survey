# 🧠 AI-Powered Survey Platform for SME Research

A custom web-based survey application designed to collect structured research data on **Artificial Intelligence (AI) capability and organizational performance** among SMEs in Lagos State.

This project integrates **web development, data collection systems, and quantitative research methodology** to support statistical analysis and academic research.

---

## 🚀 Project Overview

This platform was developed as part of my academic research project to examine:

> **“Artificial Intelligence Capability and Its Impact on Organizational Performance among SMEs in Lagos State.”**

The system enables efficient and reliable data collection from both **managers and employees**, ensuring a **multi-level organizational perspective**.

---

## ✨ Key Features

* 📊 Structured questionnaire (30+ research items)
* 👥 Multi-respondent support (Managers & Employees)
* 📱 Fully responsive design (mobile & desktop)
* 🔐 Secure data storage
* 🆔 Unique response tracking system
* 📅 Automatic timestamping of submissions
* 📤 Export data to CSV/Excel for analysis
* 📈 Designed for statistical tools like SPSS (regression, correlation, reliability)

---

## 🧾 Survey Structure

### 🔹 Section A: Demographics

* Role classification (Owner, Manager, Supervisor, Employee)
* Age, gender, industry, firm size, etc.

---

### 🔹 Section B: AI Capability Constructs

* **AI Infrastructure (AIINF)**
* **AI Human Competency (AIHC)**
* **Management Support for AI (MNSA)**
* **Data Readiness (DTR)**

Each construct is measured using multiple Likert-scale items.

---

### 🔹 Section C: Organizational Performance (OP)

Measures:

* Productivity
* Operational efficiency
* Innovation
* Decision-making quality
* Business performance

---

## 🧮 Data & Analysis

The platform is designed to produce datasets suitable for:

* Descriptive Statistics
* Reliability Testing (Cronbach’s Alpha)
* Correlation Analysis
* Multiple Linear Regression

Example model:

```
OP = β0 + β1(AIINF) + β2(AIHC) + β3(MNSA) + β4(DTR) + ε
```
